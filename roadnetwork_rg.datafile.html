roadnetwork_rg.datafile module¶

Implementation of an I/O class for WorldData used by WorldGenerator

class Datafile[source]¶

Bases: object

Implements an I/O file handler for WorldData.

Use cases:

Datafile.save(filename, key, data)
data = Datafile.load(filename, key)

file = Datafile()
file.set_data(data)
file.write(filename, key)

file = Datafile()
file.read(filename, key)
data = file2.get_data()
