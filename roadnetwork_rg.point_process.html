class PointProcess(rate, size, seed)[source]¶

Bases: Iterator[roadnetwork_rg.point_process.T]

It is an abstract class for implementing point processes.

It is a subclass of Iterator [T] that represents a stream of T type points generated by a point process. The a generic T type is on instantiation.

In order to simulate a point process _generate() should be implemented in a subclass.

__init__(rate, size, seed)[source]¶

Initializes a new iterator instance.

A private _next generator that provides the stream of points, is set by _generate().

Parameters

  • rate (RateType) – Non-homogenous processes need to have an IntensityFunction object as the value of this parameter. Homogenous processes might have int or float type (or an IntensityFunction that’s IntensityFunction.rate is 1 constantly) as the value of this.

  • size (T) – It is a T type object that describes the dimensions of the bounding box in which points are generated.

  • seed (int) – Number used initializing private pRNG instance.
