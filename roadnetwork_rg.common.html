roadnetwork_rg.common module¶

Commonly used classes and functions

class HeightMapConfig(size, height, roughness)[source]¶

Bases: object

It is a boilerplate used by HeightMap.

Parameters

  • size (int) – It is the size of height map, needs to be positive positive (corresponds to chunk_size attribute of WorldConfig). Actual value used is the smallest power of two larger or equal to this.

  • height (float) – It is the amount of displacement in first step, needs to be in [0, 1] (inclusive) range.

  • roughness (float) – It is the ratio by witch amount of displacement is changed in each step, needs to be in [0, 1] inclusive) range.

class WorldChunkData(offset_x, offset_y, height_map, cities, potential_map, pixel_paths)[source]¶

Bases: object

It is a boilerplate used by WorldChunk used by WorldGenerator.

Parameters

  • offset_x (int) – It is the x coordinate of the chunk.

  • offset_y (int) – It is the y coordinate of the chunk.

  • height_map (PIL.Image.Image) – It is a height map generated by HeightMap.generate().

  • cities (list [PointType, …]) – It is a list of coordinates generated by PointProcess.

  • potential_map (PIL.Image.Image) – It is a potential map generated by AdaptivePotentialFunction.

  • pixel_paths (dict [tuple [PointType, PointType], PixelPath]) – It is a list of coordinates generated by Pathfinder.shortest_paths()

class WorldConfig(chunk_size, height, roughness, city_rate, city_sizes=1, bit_length=64)[source]¶

Bases: object

It is a boilerplate used by WorldGenerator.

Parameters

  • chunk_size (int) – Needs to be power of 2 (see also: HeightMapConfig).

  • height (float) – (see: HeightMapConfig)

  • roughness (float) – (see: HeightMapConfig)

  • city_rate (int) – It is the expected number of cities, needs to be positive.

  • city_sizes (int) – It is the number of sizes cities can take, needs to be positive, defaults to 1.

  • bit_length (int) – Needs to be positive and even, defaults to 64 (see: HeightMap).

class WorldRenderOptions(show_debug=False, show_height_map=True, colour_height_map=True, show_cities=True, show_roads=True, show_potential_map=False)[source]¶

Bases: object

It is a boilerplate used by WorldGenerator.render().

see also: WorldGenerator.render()

Parameters

  • show_debug (bool) – When it is True information about each chunk is rendered in its upper-left corner e.g: number of cities, mean of height map …, defaults to False.

  • show_height_map (bool) – When it is True height map is rendered, defaults to True.

  • colour_height_map (bool) – When it is True the colour_palette is applied to height map before rendering it, otherwise it is rendered as grayscale image, defaults to True.

  • show_cities (bool) – When it is True cities are rendered, defaults to True.

  • show_roads (bool) – When it is True roads are rendered, defaults to True.

  • show_potential_map (bool) – When it is True final image is overlaid with potential map, defaults to False.

get_safe_seed(seed, bit_length)[source]¶

Creates a safe integer seed.

Parameters

  • seed (Any (technically)) – Only None, int, str, bytes, and bytearray are supported types.

  • bit_length (int) – Needs to be positive and less than 64.

Returns

A number safe to be used with numpy.random functions.

Raises

TypeError, ValueError
