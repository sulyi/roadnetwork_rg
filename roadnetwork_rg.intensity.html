roadnetwork_rg.intensity module¶

Various implementation of IntensityFunction

It provides some functions to be used for generating non-homogenous a PointProcess, (either using SpatialPoissonPointProcess or MarkovChainMonteCarlo method) by implementing the interface used for thinning points offered by the point process as candidates.

Bases: object

It is an abstract function for implementing composite functions.

It is used by MarkovChainMonteCarloIntensityFunction to calculate value used for thinning a point process. A composite function takes two other function a kernel and a potential function and computes a value based on them according the implemented arithmetics.
