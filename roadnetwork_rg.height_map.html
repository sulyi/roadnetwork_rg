roadnetwork_rg.height_map module¶

Terrain generator

class HeightMap(offset_x, offset_y, config, *, seed=None, bit_length=64)[source]¶

Bases: object

It is an implementation of diamond-square algorithm.

In order to allow persistent map generation uses hash based pRNG.

__init__(offset_x, offset_y, config, *, seed=None, bit_length=64)[source]¶

Initializes height map generation.

For more on parameters see also HeightMapConfig.

Parameters

  • offset_x (int) – It is the map coordinate in the x axis.

  • offset_y (int) – It is the map coordinate in the y axis.

  • config (HeightMapConfig) – It contains size, height and roughness.

  • seed (SeedType) – It is used for xor ing joined coordinates before hashing.

  • bit_length (int) – It is used for shifting y coordinate before hashing. Lower part is occupied by x coordinate.

property bit_length: int¶

It is the actual bit length used.
